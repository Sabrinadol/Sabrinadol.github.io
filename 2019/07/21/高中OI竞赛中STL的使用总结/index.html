高中STL

Content

容器

序列容器 - 有序集

vector 变长数组

关联容器 - 无序集

set 不重复元素的集合。

multiset 跟 set 具有相同功能，但允许重复的元素。

map 关联数组，每个元素含有两个数据项，map 将一个数据项映射到另一个数据项中。

容器适配器

queue 队列

priority_queue 优先队列

stack 栈

算法

binary_search 二分查找，返回值为bool，找到为真，否则为假。

upper_bound(A+1,A+n+1,v) 在有序序列A上查找大于v的第一个元素的地址。

lower_bound(A+1,A+n+1,v) 在有序序列A上查找大于等于v的第一个元素的地址。

next_permutation 全排列生成。

sort 对给定区间所有元素进行排序。

swap(a,b) 交换a和b。

unique 去除相邻的重复元素（一般需要先排序），该函数把重复的元素移到后面去了，然后依然保存到了原数组中，然后返回去重后最后一个元素的地址。
